# C语言贪吃蛇游戏（控制台版）
基于C语言 + Windows API 开发的经典贪吃蛇游戏，带用户登录注册、游戏日志、A/S调速、彩色界面。

## 功能说明
- 彩色界面（蓝墙、绿蛇、红食物）
- 用户注册/登录（user.txt 存储）
- 游戏日志记录（game_log.txt 记录时长+得分）
- A键加速、S键减速
- 方向键控制、空格暂停、L查看日志、ESC退出

## 文件说明
- snake.c：主程序源码
- user.txt：用户数据文件
- game_log.txt：游戏日志文件
- 贪吃蛇游戏软件分析与设计说明书.docx：实验报告/设计文档

## 运行环境
- Windows 系统
- Dev-C++ / Code::Blocks / Visual Studio
